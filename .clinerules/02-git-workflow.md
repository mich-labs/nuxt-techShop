# Git

Сообщения коммитов пишутся только на английском языке, в соответствии с Conventional Commits.

Пример:

```
feat(widgets): add promo-section widget with strapi mapping
fix(entities): correct link field mapping in banner mapper
```

## Флоу веток

- Для любых изменений создаётся отдельная ветка от основной ветки разработки.
- Ветка вливается через squash and merge.
- Ветка не удаляется и не мержится без явного подтверждения пользователя.

## Ограничения

- Локально можно создавать ветки и делать коммиты.
- Самостоятельный `git push` запрещён без явного запроса пользователя.
- Если при merge возникает конфликт — агент останавливается и описывает конфликт пользователю, не разрешает его самостоятельно без подтверждения.
