# Definition of Done

Перед тем как сообщить пользователю, что задача выполнена, агент проверяет каждый пункт:

- [ ] Нет нового использования `@ts-ignore`, `console.log`, `console.error`, `console.warn`.
- [ ] Если `any` использован, это инфраструктурный код, обусловленный ограничениями TypeScript/Strapi-клиента/сторонней библиотеки, и сопровождается кратким комментарием (см. `05-typescript.md`).
- [ ] Соблюдено направление импортов между слоями (`entities` не импортирует `widgets`, `shared` ничего не импортирует из вышестоящих слоёв) — см. `04-architecture-layers.md`.
- [ ] DTO-тип не "утёк" наружу слайса через `index.ts` — наружу экспортируются только Model-типы и мапперы.
- [ ] Нет `process.env` в коде, потенциально исполняемом на клиенте — использован `runtimeConfig.public`.
- [ ] Нет `:deep()`, пробивающего внутреннюю разметку чужого компонента, которым управляет не текущий слайс (см. `07-styling.md`).
- [ ] Нет дублирования markup под desktop/mobile там, где различие — это просто видимость/размер/отступ (решено через CSS/media query, не через два DOM-дерева).
- [ ] Используются существующие миксины/типы/утилиты/мапперы, а не созданы новые аналоги.
- [ ] Ответ пользователю дан на русском языке, сообщения коммитов (если создавались) — на английском в формате Conventional Commits.

Если хотя бы один пункт не выполнен — сообщи об этом явно, не скрывай отклонение от правил.
