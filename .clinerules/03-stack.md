# Стек проекта

Основной стек:

- TypeScript
- Nuxt 4 (Vue 3, файловый роутинг, SSR)
- Strapi (typed client, `~~/cms`) — источник данных
- Express API — вспомогательный/собственный бэкенд
- Sass (SCSS)
- Vite

Пакетный менеджер:

- npm

Дополнительно используется:

- Собственный миксин `device()` + карта `$breakpoints` для адаптива (см. `07-styling.md`) — не добавлять сторонние библиотеки для media queries.

Перед использованием любой библиотеки не из этого списка (включая VueUse, Pinia, UI-киты) — уточни у пользователя, не добавляй новые зависимости самостоятельно (см. критичные запреты `00-critical-and-priority.md`). Проект учебный — по умолчанию простые вещи (scroll-трекинг, media-детект и т.п.) пишутся руками через нативные Vue/Web API, а не через библиотеку, если явно не запрошено иное.
